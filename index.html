HTML

¿Qué es "HTML"

HTML (HyperText Markup Language, o Lenguaje de Marcado de Hipertexto en español) es el lenguaje estándar que se utiliza para estructurar y presentar contenido en las páginas web. Es como el esqueleto de una página web, definiendo la disposición de elementos como encabezados, párrafos, imágenes, enlaces y más.

¿Para qué sirve?

  • Organizar el contenido: HTML divide el contenido de una página en secciones y elementos, haciendo que sea más fácil de leer y comprender tanto para los usuarios como para los motores de búsqueda.
  • Crear enlaces: Permite conectar diferentes páginas web entre sí, creando la "red" que conocemos como internet.
  • Incorporar multimedia: Puedes agregar imágenes, videos, audio y otros elementos multimedia a tus páginas.
  • Crear formularios: Los formularios HTML permiten que los usuarios interactúen con la página, por ejemplo, para enviar mensajes, realizar búsquedas o completar registros.
  • Hacer que las páginas sean accesibles: HTML sigue estándares que ayudan a que las personas con discapacidades puedan acceder al contenido de una página web.

Page Structure

BODY

El <body> abarca toda la página y dentro de este la página se divide en: <Header>, <Section>, <Aside> y <Footer>.

  • HEADER

    Se ubica en la parte superior de la página y contiene la presentación de la misma, así como una barra de navegación (<nav>) que proporciona opciones para navegar por la página o hacia otras páginas.

    • NAV

      Es una barra de navegación que nos manda a distintas partes de la página o a otras páginas.

  • MAIN

    El <main> engloba el contenido principal de la página. Puede haber más de un <main> en un documento HTML, dependiendo de la estructura de la página.

    • SECTION

      El <section> se utiliza para dividir la página en diferentes secciones temáticas.

      • ARTICLE

        El <article> se usa para dividir subtemas dentro de una sección. Puede contener un encabezado (<header>) y un pie de página (<footer>).

  • ASIDE

    El <aside> representa contenido complementario o secundario. No está necesariamente ubicado al costado de un <article>, puede aparecer en diferentes partes de la página según su contenido.

  • FOOTER

    También conocido como pie de página, se ubica al final de la página o al final de una sección principal. Puede contener textos pequeños, información de copyright, enlaces y opciones menos relevantes.

EXTRAS:

  • DIV

    Es una etiqueta de bloque que se utiliza para agrupar contenido sin un significado semántico específico. Es útil para crear secciones de diseño o para estilizar elementos juntos.

  • SPAN

    Es casi lo mismo que div, es una etiqueta de línea que ocupa el espacio de su contenido. No tiene carga semántica.

  • CENTER

    solía usarse para centrar contenido, pero está obsoleta en HTML5. Se recomienda centrar contenido mediante CSS.

  • ÍNDICE

    Para crear un índice que nos traslade dentro de la página, se le da un id a los títulos y luego se enlazan mediante <a href="#id">. También puede hacerse dentro de una lista desordenada.

  • Icono:

    Para añadir un icono a la página, se utiliza <link rel="icon" href="imagen.ico">. Este enlace debe estar dentro de la sección <head> del documento HTML.

  • HR

    Se utiliza la etiqueta <hr> para añadir una línea divisoria entre contenidos.

  • HTML LANG

    Se utiliza para especificar el idioma de la página en la etiqueta HTML para mejorar el SEO y la comprensión del contenido por parte de los usuarios y motores de búsqueda.

  • BR

    Se utiliza (<br>) para añadir un salto de línea.

  • Comentar en VSCODE:

    Para comentar en Visual Studio Code se utiliza (ctrl + /).

  • Tabla de etiquetas HTML:

    Tabla periódica de las etiquetas HTML: TABLA.

  • Acordeón:

    Para hacer un acordeón se utiliza la etiqueta <details> con su respectivo título <summary>. Se puede iniciar abierto agregando el atributo open.

  • Ventana emergente:

    (Fase de experimentación) Para crear una ventana emergente se utiliza <dialog>. Por defecto permanece oculta, pero se puede hacer aparecer agregando el atributo open.

Texts

  • Las etiquetas de texto que no completan la linea:

    • Etiquetas de texto:

      • B

        Se utiliza "<b> TEXTO </b>" para que el texto este en negrita. EJ: Negrita

      • I

        Se utiliza "<i> TEXTO </i>" para que el texto este en italica/curisva. Ej: Italica / cursiva

      • U

        Se utiliza "<u> TEXTO </u>" para que el texto este subrayado. Se sugiere que se haga con CSS. Ej: Subrayado

      • STRONG

        Se utiliza "<strong> TEXTO </strong>" para indicar que el texto es importante. Ej: Texto importante

      • DEL

        Se utiliza "<del> TEXTO </del>" Para que el texto este tachado, demostrando que esta borrado. Ej: Borrado

      • S

        Se utiliza "<s> TEXTO </s>" Para que el texto este tachado, demostrando que esta incorrecto. Ej: Incorrecto

      • SMALL

        Se utiliza "<small> TEXTO </small>" para que el texto este en letra chiquita. Ej: Letra Chiquita

      • SUP

        Se utiliza "<sup> TEXTO </sup>" para agregar una superindice. Ej: e=mc2

      • SUB

        Se utiliza "<sub> TEXTO </sub>" para agregar una subindice. Ej: h2O

      • MARK

        Se utiliza "<mark> TEXTO <mark>" para resaltar textos. Ej: Marcado

      • EM

        Se utiliza "<em> TEXTO <em>" para indicar que el texto tiene énfasis Ej: Énfasis

      • CITE

        Se utiliza "<cite> TEXTO <cite>" para indicar el autor de la cita del texto. Ej: Bosch, Martin Lautaro

      • CODE

        Se utiliza "<code> TEXTO <code>" para que el texto este en formato de codigo. Ej: <link rel="stylesheet" type="text/css" href="style.css">

      • TIME

        Se utilizxa "<time> TEXTO <time>" para que el texto este en formato de fecha y hora. Ej: 19:30

  • Las etiquetas de texto que completan la linea:

    • Titulos

      • H1

      • H2:

        Titulos importantes, se pueden poner varias veces. Formula: "<h2> TITULO </h2>".

      • H3:

        Subtitulos Importantes, se pueden poner ilimitadas veces: Formula: (<h3> TITULO </h3>)

      • H4:

        Subtitulos Importantes, se pueden poner ilimitadas veces: Formula: (<h4> TITULO </h4>)

      • H5:

        Subtitulos Importantes, se pueden poner ilimitadas veces: Formula: (<h5> TITULO </h5>)

      • H6:

        Subtitulos Importantes, se pueden poner ilimitadas veces: Formula: (<h6> TITULO </h6>)

    • Etiquetas de texto:

      • BLOCKQUOTE

        Se utiliza (<blockquote> TEXTO </blockquote>) para que el texto este en formato de cita.

      • P

        Se utiliza (<p> TEXTO </p>) para crear un parrafo.

      • PRE

        Se utiliza (<pre> TEXTO <pre>) para que el navegador respete como escribiste el texto.

      • ADDRESS

        Se utiliza (<Address> </Address>) para escribir información de contacto.

Links

  • Se utiliza (<a href="Link"> Texto clickeable </a>) para redirigir a otra pagina. Ej> Click aqui.

  • Si queremos agregar una página que no sea local, tenemos que agregar el protocolo completo, por ejemplo: (https://ruta).

  • Para acceder a una carpeta, debemos especificar la ruta de la carpeta, por ejemplo: carpeta/ruta. Para volver de una carpeta, utilizamos ../ruta.

  • Si queremos abrir el link en una pestaña nueva tendremos que poner un atributo. (<a href="Link" target="_BLANK" > Texto clickeable </a>).

  • Cuando enlazamos a un sitio que no es el nuestro, es recomendable usar el atributo rel="nofollow" para indicar que ese sitio no es de nuestra propiedad. Esto es útil para el SEO.

  • Le podemos poner una imagen que apretar para que nos lleve al link deseado.

  • Para redirigir a aplicaciones podemos usar dentro del href, mailto:CORREO ELECTRONICO o tel:NUMERO DE CELULAR.

  • Para envier un mensaje en wpp usamos: href="https://api.whatsapp.com/send?phone=CELULAR&text=HOLA" para saber más hacer Click Aquí.

Lists

  • Existen dos tipso de listas, las listas desordenadas: (<ul> </ul>) y las listas ordenadas por números: (<ol> </ol>).

  • Para agregar contenidos a las listas, se utilizan los elementos de lista (<li> </li>).

  • Para especificar desde qué número debe comenzar una lista ordenada, podemos agregar el atributo "start".

  • Para invertir el orden de una lista ordenada, se utiliza el atributo "reversed".

  • Se puede cambiar la forma de enumeración de las listas utilizando el atributo "type" para especificar el tipo de enumeración deseado.

  • Para crear una lista de definición, se utiliza el elemento "<dl>" dentro del cual se colocan los términos con "<dt>" y sus definiciones con "<dd>".

  • Para mejorar la estructura de una tabla, se recomienda utilizar los elementos "<thead>" para los encabezados (títulos en negrita), "<tbody>" para el cuerpo de la tabla y "<tfoot>" para el pie de la tabla (texto descriptivo).



Multimedia

  • Para insertar una imagen se utiliza <img src="link o ruta">. Se recomienda evitar el uso de los atributos width (ancho) y height (alto) para definir el tamaño de la imagen en píxeles, ya que es preferible gestionar el tamaño mediante CSS. Esto permite mayor flexibilidad en el diseño y evita problemas de escalabilidad. Siempre se debe incluir el atributo alt para describir el contenido de la imagen, lo cual es importante para la accesibilidad, el SEO y para proporcionar información en caso de que la imagen no se cargue correctamente. Este atributo es obligatorio. También se puede utilizar el atributo title para proporcionar un título descriptivo a la imagen.

  • Se utiliza el atributo alt para describir el archivo, para el Seo, para personas con lectores de pantalla que no pueden ver, y por si pasa algo con el archivo podes describir lo que habia allí. Este atributo es obligatorio.

  • Para insertar un video se utiliza <video> </video>.

  • Para insertar un audio se utiliza <audio> </audio>.

  • Para agrupar una imagen con su respectiva descripción se puede utilizar <figure> para la imagen y <figcaption> para el texto descriptivo. Este formato proporciona un valor semántico y permite que el texto se muestre debajo de la imagen. El uso de <figcaption> es opcional.

  • Usar correctamente estos elementos y atributos no solo ayuda a mejorar la accesibilidad y la indexación en los motores de búsqueda, sino que también proporciona una estructura semántica más clara para el contenido multimedia en el documento HTML.
